# Инструкция по установке JDK на Windows

## Шаг 1: Распаковка архива JDK

1. **Создайте папку для Java** (рекомендуется):
   - Создайте папку `C:\Java\` (или `D:\Java\` если хотите на диск D)
   - Или используйте стандартную папку `C:\Program Files\Java\`

2. **Распакуйте архив JDK**:
   - Распакуйте ваш архив JDK в созданную папку
   - После распаковки у вас должна быть структура типа:
     ```
     C:\Java\jdk-17\  (или jdk-21, jdk-11 и т.д.)
       ├── bin\
       ├── lib\
       ├── include\
       └── ...
     ```

## Шаг 2: Настройка переменных окружения

### Вариант 1: Через графический интерфейс (рекомендуется)

1. Нажмите `Win + R`, введите `sysdm.cpl` и нажмите Enter
2. Перейдите на вкладку **"Дополнительно"**
3. Нажмите **"Переменные среды"**
4. В разделе **"Системные переменные"**:
   
   a. **Создайте/измените JAVA_HOME**:
      - Нажмите "Создать" (или выберите существующую и "Изменить")
      - Имя переменной: `JAVA_HOME`
      - Значение переменной: путь к папке JDK, например `C:\Java\jdk-17`
   
   b. **Измените PATH**:
      - Найдите переменную `Path` в списке
      - Нажмите "Изменить"
      - Нажмите "Создать" и добавьте: `%JAVA_HOME%\bin`
      - Или добавьте полный путь: `C:\Java\jdk-17\bin`
      - Нажмите "ОК" на всех окнах

### Вариант 2: Через PowerShell (быстрый способ)

Откройте PowerShell **от имени администратора** и выполните команды:

```powershell
# Замените путь на ваш реальный путь к JDK
$jdkPath = "C:\Java\jdk-17"

# Установка JAVA_HOME
[System.Environment]::SetEnvironmentVariable("JAVA_HOME", $jdkPath, [System.EnvironmentVariableTarget]::Machine)

# Добавление в PATH
$currentPath = [System.Environment]::GetEnvironmentVariable("Path", [System.EnvironmentVariableTarget]::Machine)
$newPath = "$jdkPath\bin;$currentPath"
[System.Environment]::SetEnvironmentVariable("Path", $newPath, [System.EnvironmentVariableTarget]::Machine)
```

## Шаг 3: Проверка установки

1. **Закройте все окна терминала/PowerShell**
2. Откройте **новый** PowerShell или командную строку
3. Выполните команды:

```powershell
java -version
javac -version
echo $env:JAVA_HOME
```

Должны увидеть что-то вроде:
```
java version "17.0.x" ...
javac 17.0.x
C:\Java\jdk-17
```

## Важные замечания

- ⚠️ После изменения переменных окружения **обязательно перезапустите** все терминалы и IDE
- ✅ Убедитесь, что путь к JDK указан правильно (без лишних пробелов, правильные слеши)
- ✅ В PATH должна быть папка `bin` внутри JDK, а не сама папка JDK

## Если что-то не работает

1. Проверьте, что папка JDK содержит подпапку `bin` с файлами `java.exe` и `javac.exe`
2. Убедитесь, что вы перезапустили терминал после изменения переменных окружения
3. Проверьте путь в переменной JAVA_HOME - он должен указывать на корневую папку JDK (не на bin!)
